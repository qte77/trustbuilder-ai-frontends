## Usage

### TrustBuilder Homepage

`make homepage_dev`

or

```sh
cd trustbuilder-page
npm install
npm run dev
```

### Wargames Suite Frontend

`make wargames_dev`

or

```sh
cd wargames-fe
npm install
npm run dev
```
